## Opis Projektu
Projekt został przygotowany w ramach przedmiotu Technologie Internetowe. Zawiera kompletną witrynę typu CV/Portfolio z dynamicznymi elementami interaktywnymi.

### Kluczowe funkcjonalności:
1. **Interaktywność (Zad 4):** Możliwość przełączania między motywem zielonym a czerwonym oraz dynamiczne sterowanie widocznością sekcji "Moje Projekty".
2. **Formularz (Zad 5):** Pełna walidacja pól: sprawdzenie pustych wartości, blokada cyfr w polu imienia oraz weryfikacja poprawności formatu e-mail.
3. **Dane Lokalne (Zad 6):** Wykorzystanie Fetch API do wczytywania listy umiejętności z zewnętrznego pliku `data.json`.
4. **Pamięć (Zad 7):** Strona zapamiętuje wybór kolorystyki użytkownika nawet po całkowitym odświeżeniu przeglądarki.
5. **API Zewnętrzne (Zad 8):** Połączenie z serwisem JSONPlaceholder w celu pobrania i wyświetlenia testowych danych z serwera w czasie rzeczywistym.

## Technologie
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Metody komunikacji:** Fetch API, AJAX
- **Przechowywanie danych:** LocalStorage, JSON
